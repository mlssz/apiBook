显示租车人附近的汽车

根据租车人与货车主的经纬度，获取租车人附近的货车主们。

REQUEST

GET

/rent/near_lessees{?uid, token, positionx, positiony, limit}
Parameters
uid number

用户id
token string

用户token
positionx number

用户经度值
positiony number

用户维度值
limit number

范围，单位米(M)

Headers
Content-Type application/json
200 application/json
